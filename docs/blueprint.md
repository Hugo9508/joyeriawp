# **App Name**: AurumLuz Digital

## Core Features:

- Interactive Home Showcase: Visually appealing landing page showcasing featured jewelry with high-quality images and engaging descriptions. Allow user to virtually 'try on' jewelry using their device's camera as a tool.
- Virtual Boutique: Online jewelry store featuring product listings, detailed filters, and an integrated WhatsApp contact button for inquiries.
- Product Details Page: In-depth view of each jewelry piece with high-resolution images, descriptions, material details, and direct WhatsApp consultation link.
- Personalized Recommendations: AI powered module which provides user specific jewelry recommendations based on past purchases and viewing history.

## Style Guidelines:

- Primary color: Gold (#D4AF37) for a luxurious feel.
- Background color: Off-white (#F8F7F6) for a clean, modern look.
- Accent color: Soft Green (#6B9080) for highlights and calls to action.
- Headline font: 'Playfair' for headings, reflecting elegance and heritage. Note: currently only Google Fonts are supported.
- Body font: 'Manrope' for body text, providing readability and a modern touch. Note: currently only Google Fonts are supported.
- Use elegant, minimalist icons to represent categories and actions.
- Incorporate subtle transitions and hover effects to enhance user experience.
